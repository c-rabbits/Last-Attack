# PC에 와이파이 없을 때 모바일로 테스트하기

PC는 유선만 있고, 폰은 다른 네트워크(데이터/다른 와이파이)에 있어도 **인터넷 터널**을 쓰면 폰에서 PC의 로컬 서버에 접속할 수 있습니다.

---

## 방법 1: ngrok (추천)

PC의 `localhost:8080`을 **임시 공개 주소**로 열어 줍니다. 폰에서 그 주소로 접속하면 됩니다.

### 1단계: ngrok 설치

- **다운로드**: https://ngrok.com/download  
  - Windows용 ZIP 받아서 풀거나  
  - Microsoft Store에서 "ngrok" 검색 후 설치

### 2단계: 가입 및 토큰 설정 (무료)

1. https://ngrok.com 에서 **Sign up** (무료 가입)
2. 가입 후 대시보드에서 **Your Authtoken** 복사
3. PowerShell 또는 CMD에서 한 번만 실행:

```bat
ngrok config add-authtoken 여기에_복사한_토큰_붙여넣기
```

### 3단계: 로컬 서버 + ngrok 실행

1. **로컬 서버**가 8080 포트로 떠 있어야 합니다. (이미 띄워 둔 상태라면 그대로 두세요.)

   ```bat
   cd e:\00_Work\02_LineMiniApp\LastAttack
   python -m http.server 8080 --bind 127.0.0.1
   ```
   (같은 PC에서만 쓸 거면 `127.0.0.1`로만 열어도 됩니다.)

2. **새 터미널**을 열고 ngrok 실행:

   ```bat
   ngrok http 8080
   ```

3. 터미널에 이런 식으로 나옵니다:
   ```
   Forwarding   https://abcd1234.ngrok-free.app -> http://localhost:8080
   ```
   이 **https://…ngrok-free.app** 주소를 **휴대폰 브라우저**에 그대로 입력하면 Last Attack이 열립니다.

- 세션을 끄면 주소는 바뀌고, 다시 `ngrok http 8080` 하면 새 주소가 나옵니다.
- 무료 버전은 한 번에 하나의 터널만 가능합니다.

---

## 방법 2: GitHub Pages에 올려서 테스트

코드를 GitHub 저장소에 푸시한 뒤 **GitHub Pages**로 배포하면, **고정된 URL**로 어디서든(폰 포함) 접속할 수 있습니다.

1. GitHub에서 **Last-Attack** 저장소 → **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: `main` (또는 사용 중인 브랜치), 폴더 `/ (root)` 선택 후 Save
4. 몇 분 뒤 `https://c-rabbits.github.io/Last-Attack/` 로 접속

- PC에 와이파이가 없어도, 폰에서 이 주소만 알면 됩니다.
- 코드 수정할 때마다 `git push` 하면 자동으로 반영됩니다.

---

## 정리

| 상황 | 추천 |
|------|------|
| 빠르게 한 번만 폰으로 확인 | **ngrok** (방법 1) |
| 자주 테스트하고 고정 주소가 좋음 | **GitHub Pages** (방법 2) |

두 방법 모두 **PC에 와이파이가 없어도** 폰에서 인터넷만 되면 테스트 가능합니다.
